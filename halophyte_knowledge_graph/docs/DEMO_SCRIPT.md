# 4-Minute Demonstration Script

## 0:00–0:30 — Project position

“Our project began with a searchable grass dictionary and two regression-based prediction modes. We extended it with a fourth phase: a research knowledge graph. It connects species with evidence from published salt-tolerance literature, rather than mixing literature claims with model estimates.”

## 0:30–1:10 — Overview

Open **Overview**. State the number of provenance records, entities, relationships and pending reviews. Explain that the included demonstration seed is intentionally small and labelled. This is a positive: it shows the system records uncertainty and evidence status instead of hiding them.

## 1:10–2:00 — Evidence explorer

Search for **Suaeda salsa**. Open its relationship with HKT1 or NHX. Point to the relationship type, exact supporting text, confidence, review status and the PubMed link. Say: “A graph edge is not just a line. It has a paper and supporting evidence attached.”

## 2:00–2:35 — Graph map

Open **Graph map**. Explain that species, mechanisms and genes form connected paths. This allows a researcher to move from a grass to a mechanism or gene and then back to the original source.

## 2:35–3:20 — Connection to the existing phases

Open the current dictionary and predictor (or describe them if using separate applications). Say: “The Phase 1 value is a dataset measurement and the Phase 2 value is a calculated estimate. This graph remains separate: it supplies literature context and citations. That prevents us from showing a prediction as if it were a published result.”

## 3:20–4:00 — Research opportunities and scale

Open **Research opportunities**. Say: “These are not discoveries being claimed by the system. They are transparent leads based on the current graph coverage. For a larger corpus, the project collects PubMed records, extracts only quoted evidence, puts uncertain claims in manual review, and audits a random sample before reporting results.”

Finish with: “This turns our application from a dataset-and-predictor project into a reproducible research-support prototype.”

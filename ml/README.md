# Machine Learning Artifacts

This folder contains the Phase 2 dataset, model outputs, and training notebook.

## Contents

- `data/halophyte_grass_library.csv` - final cleaned dataset.
- `notebooks/Phase2_Model_Training_and_Comparison.ipynb` - notebook used for model comparison.
- `models/best_model_bundle.joblib` - saved notebook model bundle.
- `models/model_comparison_results.csv` - comparison results for tested models.
- `models/model_metrics_summary.json` - evaluation summary.
- `models/numeric_fields.json` - numeric fields used for prediction.
- `requirements-notebook.txt` - notebook dependencies.

## Note

The dataset has 30 records. Predictions are estimates based on limited dataset patterns and should be presented with that limitation.

# Soil Salinity Mapping Local Streamlit App

This folder contains the original Streamlit app code extracted from
`mini-projects/soil-salinity-crop-zoning/Soil_Salinity_Mapping_FIXED.ipynb`.

The app needs the real output files generated by the notebook pipeline:

- `salinity_grid.csv`
- `pakistan_boundary.geojson`
- `target_provinces.geojson`

Run the notebook cells through the Earth Engine extraction and grid-generation
steps, then place those three files in this folder before launching:

```powershell
pip install -r integrated\soil-salinity-mapping\requirements.txt
cd integrated\soil-salinity-mapping
streamlit run app.py --server.port 8502
```

The main React app route `/soil-salinity-mapping` renders the notebook workflow
as static HTML because the Streamlit map cannot produce real predictions until
those Earth Engine-generated assets exist.
